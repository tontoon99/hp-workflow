# Plan Review Checklist

## レビュー方式
各フェーズ終了時に、`機械チェック + 人レビュー` の二段階で判定する。

## R1: HP特定レビュー
- 対象: Phase 1
- 承認者: オペレーター
- 必須確認:
  - 会社名一致
  - ドメイン一致
  - 住所/電話の一致
- 差し戻し条件:
  - 同名企業候補が2件以上
  - 確信度 0.85未満

## R2: Content Lockレビュー
- 対象: Phase 3
- 承認者: ディレクター
- 必須確認:
  - no_new_text
  - 欠落セクションなし
  - 勝手な主張追加なし
- 差し戻し条件:
  - hash不一致
  - 必須セクション欠落

## R3: 提案品質レビュー
- 対象: Phase 5
- 承認者: 制作責任者
- 必須確認:
  - Lighthouse閾値
  - 見出し構造
  - CTA導線
- 差し戻し条件:
  - accessibility < 85
  - SEO < 85

## R4: 送信レビュー
- 対象: Phase 7
- 承認者: 営業責任者
- 必須確認:
  - 宛先誤りなし
  - 送信文面に停止導線あり
  - 送信対象条件を満たす
- 差し戻し条件:
  - 停止導線欠落
  - 過去に停止希望済み
